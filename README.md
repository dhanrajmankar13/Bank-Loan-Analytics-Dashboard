# Bank Loan Analytics Dashboard (Power BI)

## Project Overview

This Power BI project analyzes bank loan data to provide insights into loan performance, customer behavior, repayment trends, and lending risk.
The dashboard helps stakeholders monitor loan applications, funded amounts, repayments, interest rates, DTI ratios, and loan quality metrics.
---
## Dashboard Features
### Summary Dashboard
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI Ratio
- Good Loan vs Bad Loan Analysis
- Loan Status Breakdown
### Overview Dashboard
- Monthly Loan Application Trends
- Loan Applications by State
- Loan Applications by Purpose
- Loan Applications by Term
- Home Ownership Analysis
---
## Key KPIs
| KPI | Description |
|------|-------------|
| Total Loan Applications | Number of loans applied |
| Total Funded Amount | Total amount funded by bank |
| Total Amount Received | Total repayments received |
| Average Interest Rate | Average loan interest rate |
| Average DTI Ratio | Debt-to-Income Ratio |

## Business Insights
- Good loans account for the majority of loan applications.
- Debt Consolidation is the most common loan purpose.
- Loan applications increase steadily throughout the year.
- Fully paid loans contribute the highest repayment volume.
- Home ownership status influences loan distribution.
